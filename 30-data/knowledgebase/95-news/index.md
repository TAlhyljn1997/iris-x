## 🚀 Submitting News

To contribute to the News Feed:

1. Create a new Markdown file: `YYYY-MM-DD-your-title.md`
2. Use the [NEWS-TEMPLATE.md](./NEWS-TEMPLATE.md) format
3. Include:
   - Clear date and category
   - Executive summary (1–3 sentences)
   - Key points
   - Technical details (if applicable)
   - Impact assessment
   - Related document links
4. Commit to the appropriate branch
5. Push to GitHub (auto-published)
