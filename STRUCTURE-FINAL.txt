🔧 TECHNICAL DETAILS
================================================================================

Framework:
  • Vanilla HTML/CSS/JavaScript (no dependencies)
  • ES6 modules (import/export)
  • Async loading (loadDocModule function)
  • Smooth scroll + fade-in animations

CSS Architecture:
  • Shared: doc-modules-styles.css (17 KB)
  • Common classes: .doc-layout, .doc-header, .doc-toc, .doc-content, .doc-section
  • Responsive: Mobile-first + @media breakpoints
  • Dark mode: prefers-color-scheme media query

JavaScript:
  • window.setLang(lang) — Language switching
  • window.setTheme(theme) — Theme switching
  • window.loadDocModule(path, exportName) — Generic module loader
  • window.returnToMainDashboard() — Return navigation
  • Automatic TOC generation from sections
  • Smooth scroll for TOC links

Performance:
  • index.html: 40 KB (cached after first load)
  • Each module: 13–28 KB (lazy loaded on demand)
  • Images: 1.6 MB total (cached)
  • Total load time: <2 seconds (typical)
